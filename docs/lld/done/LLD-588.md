### 2. technical Approach
1. **Two-Strike Rule:** Global iteration limits set to 3. Per-file implementation retries set to 2.
2. **Context Pruning:** Retries in TDD workflow now exclude previously successful files, saving ~88% of retry tokens.
3. **Telemetry:**
    - `quality.gate_rejected`: Emitted when mechanical size gate fails.
    - `retry.strike_one`: Emitted on the first retry of a file implementation.
    - `workflow.halt_and_plan`: Emitted when max iterations are reached or stagnation is detected.
