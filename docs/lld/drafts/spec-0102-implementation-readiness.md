## 1. Overview

Implement a TDD enforcement toolchain that gates implementation work behind verified failing tests, ensuring red-green-refactor discipline. The system includes a CLI tool (`tdd-gate`), git hooks (pre-commit, prepare-commit-msg), audit trail logging, and an async pending issue processor for override flows.

**Objective:** Enforce TDD discipline by gating implementation work behind verified failing tests.

**Success Criteria:** All 48 test scenarios (T010–T480) pass; pre-commit hook blocks commits without test files; `--verify-red` and `--verify-green` correctly validate test phases; override flow logs technical debt.

### 5.2 `verify_red()`

**File:** `tools/tdd_gate.py`

**Signature:**

```python
def verify_red(test_file: Path, config: TDDConfig) -> RedPhaseResult:
    """Run ONLY the specified test file and verify exit code is 1."""
    ...
```

**Input Example:**

```python
test_file = Path("tests/unit/test_tdd_gate/test_tdd_gate.py")
config = load_config()
```

**Output Example (success):**

```python
{
    "success": True,
    "test_run": {"exit_code": 1, "stdout": "FAILED test_verify_red - assert False\n1 failed in 0.42s", "stderr": "", "duration": 0.42, "test_count": 1},
    "message": "✅ Red phase verified: 1 tests failing"
}
```

**Edge Cases:**
- `test_file` doesn't exist → raises `TestFileNotFoundError(f"Test file not found: {test_file}")`
- Exit code 0 → raises `InvalidRedPhaseError("Tests passed — invalid red phase. Tests must fail first.")`
- Exit code 2 → raises `InvalidRedPhaseError("Test collection error or interrupted — not a meaningful failure.")`
- Exit code 5 → raises `InvalidRedPhaseError("No tests collected. Did you name your file test_*.py?")`
- Exit code other (e.g., 3, 4) → raises `InvalidRedPhaseError(f"Unexpected exit code {exit_code}. Check test runner output.")`
- Subprocess timeout → raises `InvalidRedPhaseError(f"Test execution timed out after 120 seconds.")`

import tempfile
import os

def save_pending_issues(issues: list[PendingIssue], queue_path: Path | None = None) -> None:
    path = queue_path or Path.home() / ".tdd-pending-issues.json"
    # Write to temp file in same directory, then rename (atomic on POSIX)
    fd, tmp_path = tempfile.mkstemp(dir=path.parent, suffix=".tmp")
    try:
        with os.fdopen(fd, "w") as f:
            json.dump(issues, f, indent=2)
        os.replace(tmp_path, path)  # Atomic rename
    except Exception:
        os.unlink(tmp_path)  # Cleanup temp on error
        raise

def _get_project_root() -> Path:
    """Get the project root directory via git."""
    try:
        result = subprocess.run(
            ["git", "rev-parse", "--show-toplevel"],
            capture_output=True, text=True, timeout=10,
        )
        if result.returncode == 0:
            return Path(result.stdout.strip())
    except (subprocess.TimeoutExpired, FileNotFoundError):
        pass
    return Path.cwd()


def load_config(config_path: Path | None = None) -> TDDConfig:
    """Load .tdd-config.json from project root or specified path.

    Falls back to sensible defaults if file not found.
    Raises: ConfigError if file exists but is malformed.
    """
    if config_path is None:
        config_path = _get_project_root() / ".tdd-config.json"

    if not config_path.exists():
        return DEFAULT_CONFIG.copy()

    try:
        with open(config_path) as f:
            data = json.load(f)
    except (json.JSONDecodeError, UnicodeDecodeError) as e:
        raise ConfigError(f"Failed to load .tdd-config.json: malformed JSON — {e}")

    # Fill in missing keys from defaults
    merged: dict[str, Any] = DEFAULT_CONFIG.copy()
    merged.update(data)
    return merged  # type: ignore[return-value]

def build_test_command(test_file: Path, config: TDDConfig) -> list[str]:
    """Construct the test command as a list of arguments."""
    base_parts = shlex.split(config["test_command"])
    framework = config["test_framework"]

    if framework == "pytest":
        return base_parts + [str(test_file), "-v", "--tb=short", "--no-header"]
    elif framework == "jest":
        return base_parts + [str(test_file), "--verbose"]
    else:
        return base_parts + [str(test_file)]


def _extract_test_count(stdout: str, framework: str) -> int:
    """Extract test count from test runner output."""
    if framework == "pytest":
        # Match patterns like "1 failed", "3 passed", "2 failed, 1 passed"
        counts = re.findall(r"(\d+) (?:passed|failed|error)", stdout)
        return sum(int(c) for c in counts) if counts else 0
    elif framework == "jest":
        # Match "Tests: N passed" or "Tests: N failed"
        match = re.search(r"Tests:\s+(\d+)", stdout)
        return int(match.group(1)) if match else 0
    return 0


def run_test_file(test_file: Path, config: TDDConfig) -> TestRunResult:
    """Execute a single test file using the configured test runner.

    Uses subprocess with list arguments (never shell=True).
    """
    command = build_test_command(test_file, config)
    start_time = datetime.now(timezone.utc)

    try:
        result = subprocess.run(
            command,
            capture_output=True,
            text=True,
            timeout=SUBPROCESS_TIMEOUT,
        )
        duration = (datetime.now(timezone.utc) - start_time).total_seconds()
        combined_output = result.stdout + result.stderr
        test_count = _extract_test_count(combined_output, config["test_framework"])

        return TestRunResult(
            exit_code=result.returncode,
            stdout=result.stdout,
            stderr=result.stderr,
            duration=duration,
            test_count=max(test_count, 0),
        )
    except subprocess.TimeoutExpired:
        duration = (datetime.now(timezone.utc) - start_time).total_seconds()
        return TestRunResult(
            exit_code=-1,
            stdout="",
            stderr=f"Test execution timed out after {SUBPROCESS_TIMEOUT} seconds.",
            duration=duration,
            test_count=0,
        )
    except FileNotFoundError as e:
        return TestRunResult(
            exit_code=-1,
            stdout="",
            stderr=f"Test runner not found: {e}",
            duration=0.0,
            test_count=0,
        )


def get_exit_code_message(exit_code: int, framework: str) -> str:
    """Return a human-readable message for a test runner exit code."""
    if framework == "jest":
        messages = EXIT_CODE_MESSAGES_JEST
    else:
        messages = EXIT_CODE_MESSAGES_PYTEST

def verify_red(test_file: Path, config: TDDConfig) -> RedPhaseResult:
    """Run ONLY the specified test file and verify exit code is 1."""
    if not test_file.exists():
        raise TestFileNotFoundError(f"Test file not found: {test_file}")

    test_run = run_test_file(test_file, config)

    if test_run["exit_code"] == 1:
        message = f"✅ Red phase verified: {test_run['test_count']} tests failing"
        return RedPhaseResult(success=True, test_run=test_run, message=message)

    if test_run["exit_code"] == -1:
        # Timeout or runner not found
        raise InvalidRedPhaseError(test_run["stderr"])

    error_msg = get_exit_code_message(test_run["exit_code"], config["test_framework"])
    raise InvalidRedPhaseError(error_msg)


def verify_green(test_file: Path, config: TDDConfig) -> GreenPhaseResult:
    """Run ONLY the specified test file and verify exit code is 0."""
    if not test_file.exists():
        raise TestFileNotFoundError(f"Test file not found: {test_file}")

    # Check for prior red phase
    state = _load_local_state()
    if state is None or state.get("red_phase") is None:
        print(
            f"⚠️ No red phase recorded for {test_file}. Run --verify-red first.",
            file=sys.stderr,
        )
        raise InvalidGreenPhaseError(
            "No red phase recorded. Run --verify-red first."
        )

    # Check that red phase was for the same test file
    red_phase = state["red_phase"]
    if red_phase and red_phase.get("test_file") != str(test_file):
        print(
            f"⚠️ Red phase was for {red_phase.get('test_file')}, not {test_file}.",
            file=sys.stderr,
        )

    test_run = run_test_file(test_file, config)

    if test_run["exit_code"] == 0:
        # Check for deleted tests
        if red_phase and test_run["test_count"] < red_phase.get("test_count", 0):
            print(
                f"⚠️ Tests deleted between red and green phase "
                f"(red: {red_phase['test_count']}, green: {test_run['test_count']})",
                file=sys.stderr,
            )
        message = f"✅ Green phase verified: {test_run['test_count']} tests passing"
        return GreenPhaseResult(success=True, test_run=test_run, message=message)

    raise InvalidGreenPhaseError("Tests still failing. Continue implementation.")

def find_test_file(source_file: Path, config: TDDConfig) -> Path | None:
    """Given a source file, find its corresponding test file.

    Searches test_dirs for files matching test_patterns.
    e.g., 'assemblyzero/utils/parser.py' → 'tests/unit/test_parser.py'
    """
    suffix = source_file.suffix
    stem = source_file.stem

    patterns = config["test_patterns"].get(suffix, [])
    if not patterns:
        return None

    project_root = _get_project_root()

    for test_dir in sorted(config["test_dirs"]):
        test_dir_path = project_root / test_dir
        if not test_dir_path.exists():
            continue

        for pattern in patterns:
            test_name = pattern.replace("{name}", stem)
            # Search directly in test_dir
            candidate = test_dir_path / test_name
            if candidate.exists():
                return candidate

            # Search recursively in test_dir subdirectories
            for match in test_dir_path.rglob(test_name):
                if match.is_file():
                    return match

    return None


def is_excluded_file(file_path: Path, config: TDDConfig) -> bool:
    """Check if a file is excluded from TDD gate by extension or path."""
    # Check extension
    if file_path.suffix in config["excluded_extensions"]:
        return True

    # Check path prefixes
    file_str = str(file_path).replace("\\", "/")
    for excluded_path in config["excluded_paths"]:
        if file_str.startswith(excluded_path):
            return True

    return False


def _is_test_file(file_path: Path, config: TDDConfig) -> bool:
    """Check if a file is itself a test file."""
    stem = file_path.stem
    suffix = file_path.suffix
    patterns = config["test_patterns"].get(suffix, [])

    for pattern in patterns:
        # Check if the file name matches a test pattern
        # For ".py" patterns: test_{name}.py → file starts with "test_"
        # For ".js" patterns: {name}.test.js → file ends with ".test"
        if pattern.startswith("test_") and stem.startswith("test_"):
            return True
        if pattern.endswith("_test" + suffix) and stem.endswith("_test"):
            return True
        if ".test." in pattern and ".test." in file_path.name:
            return True
        if ".spec." in pattern and ".spec." in file_path.name:
            return True

    return False

def handle_override(reason: str, config: TDDConfig) -> None:
    """Process --skip-tdd-gate override."""
    if not reason or not reason.strip():
        raise ValueError("Override reason cannot be empty.")

    # Import here to avoid circular dependency
    from tools.tdd_audit import append_audit_entry
    from tools.tdd_pending_issues import add_pending_issue, create_github_issue

    # Gather context
    sha = get_current_commit_sha()

    try:
        branch_result = subprocess.run(
            ["git", "rev-parse", "--abbrev-ref", "HEAD"],
            capture_output=True, text=True, timeout=10,
        )
        branch = branch_result.stdout.strip() if branch_result.returncode == 0 else "unknown"
    except (subprocess.TimeoutExpired, FileNotFoundError):
        branch = "unknown"

    try:
        remote_result = subprocess.run(
            ["git", "remote", "get-url", "origin"],
            capture_output=True, text=True, timeout=10,
        )
        remote_url = remote_result.stdout.strip() if remote_result.returncode == 0 else ""
        # Parse owner/repo from URL
        # Handles: https://github.com/owner/repo.git or git@github.com:owner/repo.git
        repo_match = re.search(r"[:/]([^/]+/[^/]+?)(?:\.git)?$", remote_url)
        repo = repo_match.group(1) if repo_match else "unknown/unknown"
    except (subprocess.TimeoutExpired, FileNotFoundError):
        repo = "unknown/unknown"

    try:
        files_result = subprocess.run(
            ["git", "diff", "--cached", "--name-only"],
            capture_output=True, text=True, timeout=10,
        )
        files_changed = [
            f.strip() for f in files_result.stdout.strip().split("\n") if f.strip()
        ] if files_result.returncode == 0 else []
    except (subprocess.TimeoutExpired, FileNotFoundError):
        files_changed = []

    # Add to pending issues queue
    pending = add_pending_issue(
        reason=reason,
        branch=branch,
        commit_sha=sha,
        repo=repo,
        files_changed=files_changed,
    )

    # Append audit entry
    state = _load_local_state()
    issue_id = state["issue_id"] if state and state.get("issue_id") else "unknown"
    append_audit_entry(
        issue_id=issue_id,
        phase="override",
        test_file=Path("N/A"),
        exit_code=-1,
        test_names=[],
        details=f"Override reason: {reason}",
        commit_sha=sha,
    )

    # Attempt async issue creation (non-blocking)
    try:
        issue_url = create_github_issue(pending)
        if issue_url:
            print(f"📋 Technical debt issue created: {issue_url}")
    except Exception:
        print(
            "⚠️ Issue creation deferred. Run 'python tools/tdd_pending_issues.py --flush' when online.",
            file=sys.stderr,
        )

    print("⚠️ TDD gate overridden. Technical debt logged.")

                phase="red",

            result = verify_green(test_file, config)

def _extract_test_names(stdout: str) -> list[str]:
    """Extract test function names from pytest output."""
    names: list[str] = []
    for line in stdout.split("\n"):
        # Match lines like "PASSED test_file.py::test_name" or "FAILED test_file.py::test_name"
        match = re.search(r"::(test_\w+)", line)
        if match:
            names.append(match.group(1))
    return names

"""TDD Audit — Append-only audit trail generation.

from __future__ import annotations

import json

def append_audit_entry(
    issue_id: str,
    phase: str,
    test_file: Path,
    exit_code: int,
    test_names: list[str],
    details: str,
    commit_sha: str | None = None,
) -> Path:
    """Append an entry to docs/reports/{issue_id}/tdd-audit.md.

    Creates file and parent directories if they don't exist.
    File is strictly append-only — never modifies existing content.
    """
    project_root = _get_project_root()
    audit_dir = project_root / "docs" / "reports" / issue_id
    audit_dir.mkdir(parents=True, exist_ok=True)
    audit_path = audit_dir / "tdd-audit.md"

    if commit_sha is None:
        commit_sha = _get_current_commit_sha()

    timestamp = datetime.now(timezone.utc).isoformat()
    test_names_str = ", ".join(test_names) if test_names else "N/A"

    # Create file with header if it doesn't exist
    if not audit_path.exists():
        header = f"# TDD Audit Trail — Issue #{issue_id}\n\n"
        audit_path.write_text(header)

    # Append entry (strictly additive)
    entry = f"""### {phase} — {timestamp}

| Field | Value |
|-------|-------|
| Phase | {phase} |
| Timestamp | {timestamp} |
| Commit | {commit_sha} |
| Test File | {test_file} |
| Exit Code | {exit_code} |
| Tests | {test_names_str} |

**Details:** {details}

---
"""
    with open(audit_path, "a") as f:
        f.write(entry)

    return audit_path


def parse_audit_file(audit_path: Path) -> list[AuditEntry]:
    """Parse a tdd-audit.md file into structured entries."""
    if not audit_path.exists():
        return []

    content = audit_path.read_text()
    entries: list[AuditEntry] = []

    # Split by "### " headers (each entry starts with ### phase — timestamp)
    sections = re.split(r"(?=^### )", content, flags=re.MULTILINE)

    for section in sections:
        section = section.strip()
        if not section.startswith("### "):
            continue

        # Parse header: ### phase — timestamp
        header_match = re.match(r"^### (\w+) — (.+)$", section, re.MULTILINE)
        if not header_match:
            continue

        phase = header_match.group(1)
        timestamp = header_match.group(2).strip()

        # Parse table fields
        commit_match = re.search(r"\| Commit \| (.+?) \|", section)
        test_file_match = re.search(r"\| Test File \| (.+?) \|", section)
        exit_code_match = re.search(r"\| Exit Code \| (.+?) \|", section)
        tests_match = re.search(r"\| Tests \| (.+?) \|", section)
        details_match = re.search(r"\*\*Details:\*\* (.+)", section)

        commit_sha = commit_match.group(1).strip() if commit_match else "unknown"
        test_file = test_file_match.group(1).strip() if test_file_match else "unknown"

        try:
            exit_code = int(exit_code_match.group(1).strip()) if exit_code_match else -1
        except ValueError:
            exit_code = -1

        tests_str = tests_match.group(1).strip() if tests_match else ""
        test_names = (
            [t.strip() for t in tests_str.split(",") if t.strip() and t.strip() != "N/A"]
            if tests_str
            else []
        )

        details_text = details_match.group(1).strip() if details_match else ""

        entries.append(
            AuditEntry(
                phase=phase,
                timestamp=timestamp,
                commit_sha=commit_sha,
                test_file=test_file,
                exit_code=exit_code,
                test_names=test_names,
                details=details_text,
            )
        )

    return entries


def generate_compliance_report(issue_id: str) -> str:
    """Generate a TDD compliance summary for a given issue."""
    project_root = _get_project_root()
    audit_path = project_root / "docs" / "reports" / issue_id / "tdd-audit.md"

    if not audit_path.exists():
        return (
            f"# TDD Compliance Report — Issue #{issue_id}\n\n"
            f"| Metric | Value |\n"
            f"|--------|-------|\n"
            f"| Verdict | ❌ NON-COMPLIANT — No audit trail found |\n"
        )

    entries = parse_audit_file(audit_path)
    if not entries:
        return (
            f"# TDD Compliance Report — Issue #{issue_id}\n\n"
            f"| Metric | Value |\n"
            f"|--------|-------|\n"
            f"| Verdict | ❌ NON-COMPLIANT — No audit entries found |\n"
        )

    red_entries = [e for e in entries if e["phase"] == "red"]
    green_entries = [e for e in entries if e["phase"] == "green"]
    override_entries = [e for e in entries if e["phase"] == "override"]

    red_str = (
        f"{red_entries[0]['timestamp']} ({red_entries[0]['commit_sha']})"
        if red_entries
        else "Not recorded"
    )
    green_str = (
        f"{green_entries[0]['timestamp']} ({green_entries[0]['commit_sha']})"
        if green_entries
        else "Not recorded"
    )

    # Calculate duration
    duration_str = "N/A"
    if red_entries and green_entries:
        try:
            red_ts = datetime.fromisoformat(red_entries[0]["timestamp"])
            green_ts = datetime.fromisoformat(green_entries[0]["timestamp"])
            delta = green_ts - red_ts
            hours, remainder = divmod(int(delta.total_seconds()), 3600)
            minutes = remainder // 60
            duration_str = f"{hours}h {minutes}m" if hours else f"{minutes}m"
        except (ValueError, TypeError):
            duration_str = "Unable to calculate"

    override_str = "Yes" if override_entries else "No"

    # Determine verdict
    if override_entries:
        verdict = "⚠️ OVERRIDE — TDD gate was overridden. See audit trail."
    elif red_entries and green_entries:
        verdict = "✅ COMPLIANT"
    elif red_entries:
        verdict = "⏳ IN PROGRESS — Red phase verified, awaiting green phase"
    else:
        verdict = "❌ NON-COMPLIANT"

    return (
        f"# TDD Compliance Report — Issue #{issue_id}\n\n"
        f"| Metric | Value |\n"
        f"|--------|-------|\n"
        f"| Red Phase | {red_str} |\n"
        f"| Green Phase | {green_str} |\n"
        f"| Red→Green Duration | {duration_str} |\n"
        f"| Override Used | {override_str} |\n"
        f"| Verdict | {verdict} |\n"
    )

    branch: str
    commit_sha: str
    repo: str
    files_changed: list[str]
    status: str
    issue_url: str | None
    retry_count: int


class FlushResult(TypedDict):
    """Result from flushing pending issues."""

    created_count: int
    failed_count: int
    remaining_count: int


DEFAULT_QUEUE_PATH = Path.home() / ".tdd-pending-issues.json"


def load_pending_issues(queue_path: Path | None = None) -> list[PendingIssue]:
    """Load pending issues from ~/.tdd-pending-issues.json."""
    path = queue_path or DEFAULT_QUEUE_PATH
    if not path.exists():
        return []
    try:
        with open(path) as f:
            data = json.load(f)
        if not isinstance(data, list):
            return []
        return data
    except (json.JSONDecodeError, UnicodeDecodeError):
        return []


def save_pending_issues(
    issues: list[PendingIssue], queue_path: Path | None = None,
) -> None:
    """Save pending issues back to ~/.tdd-pending-issues.json.

    Atomically writes (write to temp, then rename).
    """
    path = queue_path or DEFAULT_QUEUE_PATH
    path.parent.mkdir(parents=True, exist_ok=True)

    fd, tmp_path = tempfile.mkstemp(dir=path.parent, suffix=".tmp")
    try:
        with os.fdopen(fd, "w") as f:
            json.dump(issues, f, indent=2)
        os.replace(tmp_path, str(path))
    except Exception:
        try:
            os.unlink(tmp_path)
        except OSError:
            pass
        raise


def add_pending_issue(
    reason: str,
    branch: str,
    commit_sha: str,
    repo: str,
    files_changed: list[str],
    queue_path: Path | None = None,
) -> PendingIssue:
    """Add a new pending issue to the queue."""
    issues = load_pending_issues(queue_path)

    new_issue = PendingIssue(
        timestamp=datetime.now(timezone.utc).isoformat(),
        reason=reason,
        branch=branch,
        commit_sha=commit_sha,
        repo=repo,
        files_changed=files_changed,
        status="pending",
        issue_url=None,
        retry_count=0,
    )

    issues.append(new_issue)
    save_pending_issues(issues, queue_path)
    return new_issue


def check_gh_auth() -> bool:
    """Check if gh CLI is authenticated."""
    try:
        result = subprocess.run(
            ["gh", "auth", "status"],
            capture_output=True, text=True, timeout=10,
        )
        return result.returncode == 0
    except (subprocess.TimeoutExpired, FileNotFoundError):
        return False


def create_github_issue(pending: PendingIssue) -> str | None:
    """Create a single GitHub issue via gh CLI.

    Uses subprocess.run with list arguments (never shell=True).
    """
    title = f"TDD Debt: {pending['branch']} — override used"

    files_str = "\n".join(f"- `{f}`" for f in pending["files_changed"]) or "- (no files recorded)"

    body = (
        f"## TDD Override\n\n"
        f"**Reason:** {pending['reason']}\n\n"
        f"**Branch:** `{pending['branch']}`\n\n"
        f"**Commit:** `{pending['commit_sha']}`\n\n"
        f"**Timestamp:** {pending['timestamp']}\n\n"
        f"**Files Changed:**\n{files_str}\n\n"
        f"---\n"
        f"*Auto-generated by TDD Gate override flow (Issue #102)*"
    )

    try:
        result = subprocess.run(
            [
                "gh", "issue", "create",
                "--title", title,
                "--body", body,
                "--label", "tech-debt,tdd-override",
                "--repo", pending["repo"],
            ],
            capture_output=True, text=True, timeout=30,
        )

        if result.returncode == 0:
            # gh outputs the issue URL on stdout
            issue_url = result.stdout.strip()
            return issue_url
        return None
    except (subprocess.TimeoutExpired, FileNotFoundError):
        return None


def flush_pending_issues(
    queue_path: Path | None = None, max_retries: int = 3,
) -> FlushResult:
    """Attempt to create all pending GitHub issues via gh issue create."""
    issues = load_pending_issues(queue_path)
    pending = [i for i in issues if i["status"] == "pending"]

    if not pending:
        print("No pending issues to flush.")
        return FlushResult(created_count=0, failed_count=0, remaining_count=0)

    if not check_gh_auth():
        print(
            "❌ gh CLI not authenticated. Run 'gh auth login' first.",
            file=sys.stderr,
        )
        return FlushResult(
            created_count=0, failed_count=0, remaining_count=len(pending),
        )

    created_count = 0
    failed_count = 0

    for issue in issues:
        if issue["status"] != "pending":
            continue

        issue_url = create_github_issue(issue)
        if issue_url:
            issue["status"] = "created"
            issue["issue_url"] = issue_url
            created_count += 1
            print(f"✅ Created: {issue_url}")
        else:
            issue["retry_count"] += 1
            if issue["retry_count"] >= max_retries:
                failed_count += 1
                print(
                    f"❌ Failed after {max_retries} attempts: "
                    f"TDD Debt: {issue['branch']}",
                    file=sys.stderr,
                )

    save_pending_issues(issues, queue_path)

    remaining = len([i for i in issues if i["status"] == "pending"])
    result = FlushResult(
        created_count=created_count,
        failed_count=failed_count,
        remaining_count=remaining,
    )
    print(
        f"\nSummary: Created: {created_count}, "
        f"Failed: {failed_count}, Remaining: {remaining}",
    )
    return result


def main() -> int:
    """CLI entry point for pending issues processor."""
    parser = argparse.ArgumentParser(
        description="TDD Pending Issues — process override queue",
        prog="tdd-pending-issues",
    )

    group = parser.add_mutually_exclusive_group(required=True)
    group.add_argument(
        "--flush",
        action="store_true",
        help="Attempt to create all pending GitHub issues",
    )
    group.add_argument(
        "--list",
        action="store_true",
        help="List all pending issues",
    )

    parser.add_argument(
        "--max-retries",
        type=int,
        default=3,
        help="Maximum retry attempts per issue (default: 3)",
    )
    parser.add_argument(
        "--queue-path",
        help="Path to pending issues JSON file",
    )

    args = parser.parse_args()
    queue_path = Path(args.queue_path) if args.queue_path else None

    if args.flush:
        result = flush_pending_issues(queue_path, args.max_retries)
        return 0 if result["failed_count"] == 0 else 1
    elif args.list:
        issues = load_pending_issues(queue_path)
        pending = [i for i in issues if i["status"] == "pending"]
        if not pending:
            print("No pending issues.")
            return 0
        for i, issue in enumerate(pending, 1):
            print(
                f"{i}. [{issue['timestamp']}] {issue['branch']} — {issue['reason']}"
                f" (retries: {issue['retry_count']})",
            )
        return 0

    return 0


if __name__ == "__main__":
    sys.exit(main())

"""Pre-commit hook: verify test file exists for changed implementation files.

Issue #102: TDD Test Initialization Gate

Invoked by .husky/pre-commit. Returns 0 to allow commit, 1 to block.
Fail-open: configuration errors allow the commit with a warning.
"""

from __future__ import annotations

import subprocess
import sys
from pathlib import Path


def _get_project_root() -> Path:
    """Get the project root directory via git."""
    try:
        result = subprocess.run(
            ["git", "rev-parse", "--show-toplevel"],
            capture_output=True, text=True, timeout=10,
        )
        if result.returncode == 0:
            return Path(result.stdout.strip())
    except (subprocess.TimeoutExpired, FileNotFoundError):
        pass
    return Path.cwd()


def get_staged_files() -> list[Path]:
    """Get list of files staged for commit.

    Uses: git diff --cached --name-only --diff-filter=ACM
    Returns: List of Path objects for added/copied/modified files.
    """
    try:
        result = subprocess.run(
            ["git", "diff", "--cached", "--name-only", "--diff-filter=ACM"],
            capture_output=True, text=True, timeout=10,
        )
        if result.returncode != 0:
            return []
        return [
            Path(f.strip())
            for f in result.stdout.strip().split("\n")
            if f.strip()
        ]
    except (subprocess.TimeoutExpired, FileNotFoundError):
        return []


def filter_source_files(files: list[Path], config: dict) -> list[Path]:
    """Filter file list to only source code files that require tests.

    Excludes: test files themselves, docs, config, excluded paths.
    """
    # Import here to use shared logic
    sys.path.insert(0, str(_get_project_root()))
    try:
        from tools.tdd_gate import is_excluded_file, _is_test_file
    except ImportError:
        # If tdd_gate can't be imported, allow the commit
        return []

    source_files = []
    for f in files:
        if is_excluded_file(f, config):
            continue
        if _is_test_file(f, config):
            continue
        # Only check files in source_dirs
        file_str = str(f).replace("\\", "/")
        in_source_dir = any(
            file_str.startswith(sd) for sd in config.get("source_dirs", [])
        )
        if in_source_dir:
            source_files.append(f)

    return source_files


def pre_commit_check() -> int:
    """Main pre-commit hook entry point.

    Fail-open: on any error, prints warning and returns 0.
    """
    try:
        sys.path.insert(0, str(_get_project_root()))
        from tools.tdd_gate import load_config, find_test_file

        config = load_config()
    except Exception as e:
        print(
            f"⚠️ TDD Gate: Configuration error ({e}), allowing commit.",
            file=sys.stderr,
        )
        return 0

    staged_files = get_staged_files()
    if not staged_files:
        return 0

    source_files = filter_source_files(staged_files, config)
    if not source_files:
        return 0

    blocked_files: list[tuple[Path, list[str]]] = []

    for source_file in source_files:
        test_path = find_test_file(source_file, config)
        if test_path is None:
            # Build expected test file names for the error message
            stem = source_file.stem
            suffix = source_file.suffix
            patterns = config.get("test_patterns", {}).get(suffix, [])
            expected = [
                p.replace("{name}", stem) for p in patterns
            ]
            blocked_files.append((source_file, expected))

    if blocked_files:
        print("\n❌ TDD Gate: No test file found for the following source files:\n")
        for source_file, expected in blocked_files:
            expected_str = " or ".join(expected)
            test_dirs_str = ", ".join(config.get("test_dirs", []))
            print(f"  {source_file}")
            print(f"    Expected: {expected_str} in {test_dirs_str}")
            print()
        print(
            'Commit blocked. Write tests first, or use '
            '--skip-tdd-gate --reason "..."\n'
        )
        return 1

    return 0


if __name__ == "__main__":
    sys.exit(pre_commit_check())

"""Prepare-commit-msg hook: append TDD phase footers to commit messages.

Issue #102: TDD Test Initialization Gate

Invoked by .husky/prepare-commit-msg with args: <commit-msg-file> [<source>] [<sha>]
Runs BEFORE editor opens and BEFORE GPG signing.
Returns 0 always (does not block commits).
"""

from __future__ import annotations

import json
import subprocess
import sys
from pathlib import Path
from typing import Any


def _get_project_root() -> Path:
    """Get the project root directory via git."""
    try:
        result = subprocess.run(
            ["git", "rev-parse", "--show-toplevel"],
            capture_output=True, text=True, timeout=10,
        )
        if result.returncode == 0:
            return Path(result.stdout.strip())
    except (subprocess.TimeoutExpired, FileNotFoundError):
        pass
    return Path.cwd()


def read_local_state(state_path: Path | None = None) -> dict[str, Any] | None:
    """Read .tdd-state.json from project root.

    Returns None if file doesn't exist or is malformed.
    """
    if state_path is None:
        state_path = _get_project_root() / ".tdd-state.json"
    if not state_path.exists():
        return None
    try:
        with open(state_path) as f:
            return json.load(f)
    except (json.JSONDecodeError, UnicodeDecodeError, OSError):
        return None


def append_footer(
    commit_msg_file: Path, footer_key: str, footer_value: str,
) -> None:
    """Append a git trailer/footer to a commit message file.

    Ensures proper formatting with blank line separator before trailers.
    Does not duplicate existing footers.
    """
    content = commit_msg_file.read_text()

    # Check for duplicate
    if f"{footer_key}:" in content:
        return

    # Ensure blank line before footer
    if not content.endswith("\n\n"):
        if content.endswith("\n"):
            content += "\n"
        else:
            content += "\n\n"

    content += f"{footer_key}: {footer_value}\n"
    commit_msg_file.write_text(content)


def prepare_commit_msg(
    commit_msg_file: Path, commit_source: str | None = None,
) -> int:
    """Append TDD footer to commit message if red/green phase is verified.

    Returns: 0 always (does not block commits).
    """
    # Skip merge and squash commits
    if commit_source in ("merge", "squash"):
        return 0

    state = read_local_state()
    if state is None:
        return 0

    red_phase = state.get("red_phase")
    if red_phase and isinstance(red_phase, dict):
        sha = red_phase.get("commit_sha", "unknown")
        ts = red_phase.get("timestamp", "unknown")
        append_footer(commit_msg_file, "TDD-Red-Phase", f"{sha}:{ts}")

    green_phase = state.get("green_phase")
    if green_phase and isinstance(green_phase, dict):
        sha = green_phase.get("commit_sha", "unknown")
        ts = green_phase.get("timestamp", "unknown")
        append_footer(commit_msg_file, "TDD-Green-Phase", f"{sha}:{ts}")

    return 0


def main() -> int:
    """Entry point called by git prepare-commit-msg hook."""
    try:
        if len(sys.argv) < 2:
            return 0

        commit_msg_file = Path(sys.argv[1])
        commit_source = sys.argv[2] if len(sys.argv) > 2 else None

        return prepare_commit_msg(commit_msg_file, commit_source)
    except Exception as e:
        # Fail-open: never block commits
        print(f"⚠️ TDD prepare-commit-msg hook error: {e}", file=sys.stderr)
        return 0


if __name__ == "__main__":
    sys.exit(main())

## 10. Implementation Notes

### 10.1 Error Handling Convention

All functions follow a fail-open philosophy for hooks (pre-commit, prepare-commit-msg) — errors are caught at the top level and the hook returns 0 with a warning. CLI commands (`--verify-red`, `--verify-green`) fail explicitly with specific error messages and non-zero exit codes.

Custom exceptions are defined in `tools/tdd_gate.py`:
- `ConfigError` — malformed configuration
- `TestFileNotFoundError` — test file doesn't exist on disk
- `InvalidRedPhaseError` — red phase verification failed
- `InvalidGreenPhaseError` — green phase verification failed

### 10.2 Logging Convention

CLI output uses emoji prefixes for quick scanning:
- `✅` — Success (phase verified)
- `❌` — Error (blocking failure)
- `⚠️` — Warning (non-blocking)
- `📋` — Info (issue created)

Error output goes to `sys.stderr`. Success output goes to `sys.stdout`.

### 10.3 Constants

| Constant | Value | Rationale |
|----------|-------|-----------|
| `SUBPROCESS_TIMEOUT` | `120` | Maximum seconds for test execution before timeout |
| `MAX_FAILURE_OUTPUT` | `2000` | Truncate failure output to 2000 chars in state file |
| `DEFAULT_MAX_RETRIES` | `3` | Maximum issue creation retry attempts |
| `GH_TIMEOUT` | `30` | Timeout for `gh issue create` subprocess call |

### 10.4 Import Strategy for Circular Dependencies

`tools/tdd_gate.py` imports from `tools/tdd_audit.py` and `tools/tdd_pending_issues.py` using lazy imports (inside functions, not at module top level) to avoid circular import issues and to keep the module importable even when audit/pending modules are not yet available.

```python
# In handle_override() and main():
from tools.tdd_audit import append_audit_entry
from tools.tdd_pending_issues import add_pending_issue, create_github_issue
```

### 10.5 Path Resolution

All paths are resolved relative to the project root (obtained via `git rev-parse --show-toplevel`). The `_get_project_root()` helper is defined in each module rather than imported, to keep each module independently functional.

### 10.6 Husky Setup

After modifying `dashboard/package.json`, run from the `dashboard/` directory:

```bash
cd dashboard
npm install
```

This triggers the `prepare` script which runs `cd .. && husky`, initializing the `.husky/` directory at the repo root. The `.husky/pre-commit` and `.husky/prepare-commit-msg` files should already exist from this implementation.
