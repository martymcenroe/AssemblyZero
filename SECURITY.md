# Security Policy

## Reporting a Vulnerability

If you discover a security vulnerability in this project, please report it responsibly.

**Report via:** [Open a security issue](https://github.com/martymcenroe/AssemblyZero/issues/new)

## Supported Versions

| Version | Supported |
|---------|-----------|
| main    | Yes       |

## Scope

This project includes tools that interact with GitHub APIs, local filesystems, and shell environments. Security issues in any of these areas are in scope.
